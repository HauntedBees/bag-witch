# Bag Witch

*Bag Witch the Bog Witch* was developed between July 7th and 20th, 2026, for the [Summer Slow Jams 2026: Inventory](https://itch.io/jam/ssjinventory) game jam hosted by the [Portland Indie Game Squad](https://pigsquad.itch.io/). You can play it, for free, on [itch.io](https://hauntedbees.itch.io/bag-witch)!
